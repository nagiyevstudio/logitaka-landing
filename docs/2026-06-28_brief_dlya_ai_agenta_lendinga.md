# Бриф для ИИ-агента лендинга: Voice-first wedge и международный рынок

## Контекст стратегического пивота

Logitaka переходит на **английский как единственный публичный язык** продукта и продвижения. Мы выходим на международный SaaS-рынок, отказываясь от русского и азербайджанского языков в публичном поле.

**Причина:** СНГ рынок сложен для монетизации комплексного SaaS (культурно непривычно платить за софт), тогда как англоязычная аудитория воспринимает подписку на productivity/AI-инструменты как норму. Платёжный контур (MoR через Lemon Squeezy, USD) уже почти готов.

## Стратегия "острого клина" (Wedge Strategy)

Вместо продвижения всех возможностей Logitaka одновременно, мы выходим на рынок через **одну конкретную фичу** — голосовой ассистент (Voice-to-Structure).

### Что это значит:
- **Публичное позиционирование:** только голосовой ассистент, который принимает неструктурированный голосовой ввод (планы, решения, задачи, события) и автоматически раскладывает его по полочкам: план, календарь, контекст проекта
- **Второй контур (не продвигается на витрине):** обсуждения, мультиагентная система, PRO-инструменты — остаются внутри продукта для удержания и апгрейда, но не являются главным сообщением
- **Лендинг, SMM и onboarding:** говорят только о Voice-to-Structure

### Почему голосовой ввод:
- Самая визуально демонстративная и вирусная фича: 15-секундный ролик показывает результат без объяснения архитектуры
- Боль универсальна для founders, consultants, neurodivergent professionals
- Низкий барьер входа обеспечивает viral loop через Starter-план

## Требования к SEO и AEO (Answer Engine Optimization)

### SEO (Search Engine Optimization):

**Ключевые запросы для таргетинга:**
- "voice task manager"
- "AI voice assistant for productivity"
- "voice to task app"
- "AI project management voice"
- "voice-first productivity tool"
- "cognitive exoskeleton for work"
- "AI context retention"
- "voice-powered task management"

**Структурные требования:**
- Семантический HTML5 (header, main, section, article)
- Schema.org разметка: SoftwareApplication, Product, FAQPage
- Meta-теги: title, description, Open Graph, Twitter Cards
- Alt-тексты для всех изображений
- Внутренняя перелинковка между секциями
- Мобильная адаптивность (mobile-first)
- Скорость загрузки < 3 секунд
- HTTPS обязательно

**Контентные требования:**
- H1-H6 иерархия заголовков
- Ключевые слова в первых 100 словах
- LSI-связанные термины (voice input, task automation, context management, AI-powered)
- FAQ-секция с структурированными ответами
- Case studies / use cases с конкретными результатами

### AEO (Answer Engine Optimization):

**Для AI-поисковиков (ChatGPT, Perplexity, Claude, Gemini):**

1. **Структурированные данные:**
   - JSON-LD с FAQ schema
   - HowTo schema для "How it works"
   - Product schema с pricing

2. **Ответы в формате вопрос-ответ:**
   - "What is Logitaka?" — четкий одноабзацный ответ
   - "How does voice-to-structure work?" — пошаговое объяснение
   - "Who is Logitaka for?" — конкретные портреты
   - "How much does it cost?" — прозрачный pricing

3. **Контекстные маркеры:**
   - Явные определения терминов
   - Сравнения с альтернативами (Notion, Todoist, ChatGPT)
   - Конкретные use cases с ролями (solo founder, PM, consultant)

4. **Цитируемость:**
   - Короткие, фактологические предложения
   - Статистика и цифры (10+ AI models, 3 languages, 24/7)
   - Уникальные формулировки ("cognitive exoskeleton", "voice-first productivity OS")

5. **Entity recognition:**
   - Явное упоминание интеграций (OpenAI, Anthropic, Google, Lemon Squeezy)
   - География (international, USD pricing)
   - Тип продукта (SaaS, web app, PWA)

## Технические ограничения

1. **Задержка голосового ассистента:** 3+ секунды из-за маршрутизации между моделями. Решается не бэкендом, а UX-паттернами:
   - Прогрессивное раскрытие результатов
   - Микроанимации обработки
   - Стриминг при long-form записи
   - Оптимистичный UI

2. **Мультиязычность внутри приложения:** сохраняется (пользователь может выбрать язык в профиле), но лендинг — только английский.

3. **Платежи:** Lemon Squeezy (MoR), USD, автопродление. Никакого ручного биллинга. (об этом можно не писать в лендинге. это просто для инфы)
